# Enryh Wiki

欢迎来到 Enryh Wiki！这是一个基于 VuePress 构建的文档网站。

## 特性

- 🚀 基于 VuePress 2.x
- 📱 响应式设计
- 🔍 内置搜索功能
- 🎯 清晰的导航结构
- ⚡ 快速加载

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建生产版本
npm run build
```

## 目录结构

```
docs/
├── .vuepress/
│   ├── config.ts    # 配置文件
│   └── public/      # 静态资源
├── guide/           # 指南文档
├── api/            # API 文档
└── README.md       # 主页
```
